# EasyGelAlyzer

**EasyGelAlyzer** は、ゲル電気泳動画像から簡易的に定量解析を行う軽量なデスクトップアプリケーションです。タンパク質（SDS-PAGE）と DNAの両モードに対応しており、バンド位置の測定・Rf 値の算出・検量線作成・分子量（または塩基対サイズ）の推定をひとつの GUI で完結できます。

---

## 主な機能

* **デュアル解析モード** — タンパク質（SDS-PAGE、kDa）と DNA（アガロースゲル、bp）を選択可能
* **インタラクティブ画像ビューア** — ゲル画像のズーム・パン・回転、輝度・コントラスト調整に加え、**背景補正（Rolling-ballアルゴリズム）**をサポート
* **便利なズーム・位置リセット** — ダブルクリック、中ボタンのダブルクリック、または Shift キーの2回押しで、画像の表示倍率と位置を瞬時に自動調整（トリミング後のサイズにも追従）
* **開始/終了ライン設定** — クリックで泳動距離の基準ラインを配置（後からドラッグで微調整可能）
* **分子量マーカー登録** — マーカーバンドをクリックして既知サイズを入力すると検量線が自動更新（手動入力のほか、プリセットからの選択も可能）
* **分子量マーカー表示の改善** — ラベルを画像外側に表示して操作の干渉を防止し、文字列を2行に分けて中央揃えにすることで視認性を向上
* **試料バンド測定** — 試料バンドをクリックすると検量線からサイズを自動推定（マーカー一括非表示中でも測定可能）
* **検量線自動作成** — log₁₀(サイズ) vs. Rf の線形回帰と R² 表示；係数の手動上書きも可能
* **レーンラベルとドラッグ操作の改善** — ゲル画像上部にドラッグ可能なレーンラベルを追加。ドラッグ時は中心付近で軽くスナップし、ガイドラインが表示されるため位置合わせが容易
* **レイヤーパネルの改善** — アノテーション項目ごとに表示／出力可否をトグルでき、名前変更・並び替え・削除をサポート。スクロールバーが追加され、アノテーション数が多くても快適に操作可能
* **多様なエクスポート機能** — マーカーデータ・試料結果・検量線グラフを含む **Excel** (`.xlsx`)、**CSV** (`.csv`)、またはクリップボードコピーに対応
* **高画質アノテーション画像エクスポート** — 全アノテーションを描画した画像を保存（複数レイアウト、白黒モード対応）。低解像度画像（短辺1200px・長辺1800px未満）は自動で最大4倍まで高解像度化し、斜め線にはアンチエイリアス処理を適用するため、ギザつき（ジャギー）のない滑らかな文字や線で出力可能
* **プロジェクト保存 / 読み込み** — 全計測データと画像を `.gelproj` JSON ファイルに保存。**起動時ダイアログからの直接読み込み**にも対応し、スムーズに再開可能
* **スマートな保存確認** — 未保存または変更があった場合のみ確認ダイアログを表示（既存プロジェクトがある場合は上書き保存として処理）
* **やり直し / 元に戻す** — 画像操作やアノテーション状態の 20 ステップ履歴
* **ドラッグ＆ドロップ** — 画像ファイルをキャンバスに直接ドロップして読み込み可能（`tkinterdnd2` 要）
* **高速な起動** — Matplotlib / NumPy などの重いライブラリを遅延読み込み（Lazy Load）に変更し、起動時間を約1.46秒から約0.25秒へと大幅に高速化
* **日本語・英語対応 UI** — 起動時に英語 / 日本語を切り替え可能（起動後も動的に切替可能）

## インストール

### 方法 A — ビルド済み実行ファイルを使う（推奨）

[Releases](../../releases) ページから最新の `EasyGelAlyzer_バージョン.zip` をダウンロードし、解凍後、`dirt/EasyGelAlyzer.exe` か `launcer.exe` をダブルクリックで起動します。Python や追加ライブラリのインストールは不要です。

### 方法 B — ソースコードからビルドする

リポジトリをクローンして自分で実行ファイルを生成する場合：

1. **必要な環境を準備する**

   以下の環境が必要です：

   * **Python 3.9 以上**
   * **Windows（PyInstaller の仕様上）**

   必要な Python ライブラリをインストールします：

   ```bash
   pip install pillow numpy matplotlib openpyxl pyinstaller
   pip install tkinterdnd2   # ドラッグ＆ドロップ対応（任意）
   ```

2. **ビルドスクリプトを実行する**

   リポジトリのルートにある `exe_build.bat` を実行します。

   * ダブルクリックで実行
   * またはターミナルから：

   ```bat
     exe_build.bat
     ```

3. **ビルドスクリプトが行う処理**

   `exe_build.bat` は以下の処理を自動で行います：

   * `src/version.py` からバージョン番号を読み取る
   * Git が利用可能な場合：

     * `git add .`
     * `git commit -m "Build vX.Y.Z"`
     * `git tag vX.Y.Z`
   * 日本語パスによる PyInstaller の不具合を避けるため、TEMP ディレクトリに一時ビルド環境を作成
   * PyInstaller を実行し、以下をバンドル：

     * `src/main.py`
     * `src/assets/`（アイコンなど）
   * 完成した `EasyGelAlyzer.exe` をローカルの `dist/` フォルダにコピー
   * `EasyGelAlyzer.lnk`（ショートカット）を自動生成
   * 一時ビルドフォルダを削除してクリーンアップ
4. **ビルド結果**

   ビルドが成功すると、以下が生成されます：

   * `dist/EasyGelAlyzer.exe`
   * `EasyGelAlyzer.lnk`（ショートカット）

   これらを使ってアプリを起動できます。

5. **注意事項**

   * `src/version.py` が存在しない場合、ビルドは中断されます
   * 日本語パス環境でも確実にビルドできるよう設計されています
   * `src/assets/` 内のアイコンは自動で exe に埋め込まれます
   * Git がインストールされていない場合はコミット・タグ付けをスキップします

## クイックスタート

1. **起動** — `EasyGelAlyzer.exe` をダブルクリックします（ソースからビルドする場合は先に `exe_build.bat` を実行してください）。
2. **モード選択** — 起動ダイアログで「タンパク質（kDa）」、「DNA（bp）」を選択するか、「プロジェクトファイル読み込み」から既存の `.gelproj` を読み込んで再開します。
3. **画像を開く** — *ファイル → 画像を開く* から、または JPEG / PNG / TIFF 形式のゲル画像をキャンバスにドラッグ＆ドロップして読み込みます。
4. **画像調整・背景補正** — 必要に応じて *画像調整* で明るさやコントラストを変更したり、*背景補正* を適用してノイズや背景のムラを取り除きます。
5. **基準ラインの設定** — *開始ライン設定* をクリックしてゲルの上端（Rf = 0）をクリック、次に *終了ライン設定* で色素前線や泳動の下端（Rf = 1）をクリックします。
6. **マーカーを追加** — *マーカー追加* をクリックしてから各マーカーバンドをクリックし、既知サイズを入力します（手動のほか、プリセット登録機能も利用可能です。Escで終了）。
7. **検量線を確認** — 検量線パネルの R² 値を確認します（R² ≥ 0.95 が推奨）。手動で係数を上書きすることも可能です。
8. **試料を追加** — *試料追加* をクリックしてから各試料バンドをクリックすると、サイズが自動推定されます。レーンラベルを配置する場合は、*ラベル追加* を押したあとにドラッグで位置調整します（中心付近で自動的にスナップします）。
9. **エクスポート** — 表データは *エクスポート → Excel* または *CSV*、あるいは結果テーブル下のボタンからクリップボードにコピー可能です。画像は *エクスポート → アノテーション画像* を使用します。
10. **保存** — *ファイル → プロジェクト保存* (`.gelproj`）で全データを保存します。既存プロジェクトを開いている場合は終了時に自動的に上書き保存の確認が行われます。

## 検量線の計算方法

Rf 値は以下の式で算出されます：

```
Rf = (バンドの y 座標 − 開始ラインの y 座標) / (終了ラインの y 座標 − 開始ラインの y 座標)
```

次に線形回帰を行います：

```
log₁₀(サイズ) = a × Rf + b
```

未知サイズの推定は以下の式で行います：

```
サイズ = 10^(a × Rf + b)
```

マーカーがない場合は、検量線パネルで係数 `a` と `b` を手動入力することもできます。

## エクスポートの詳細

### Excel（`.xlsx`）

3 枚のシートが生成されます：

* **検量線シート** — マーカー名、Rf、サイズ、log₁₀(サイズ)
* **結果シート** — 試料番号、試料名、Rf、推定サイズ
* **グラフシート** — 検量線グラフ画像を埋め込み

### CSV（`.csv`）

表形式の測定データ（試料番号、試料名、Rf、推定サイズ）をシンプルに CSV ファイルとして出力します。

### クリップボードコピー

測定結果テーブル下部のボタンから、結果データを直接クリップボードにコピーできます。スプレッドシートやドキュメントへの貼り付けに便利です。

### アノテーション付き画像

4 種のレイアウトから選択し出力できます。

* **自動高解像度化**: 低解像度の画像（短辺1200px未満または長辺1800px未満）を出力する際、自動で最大4倍まで高解像度化してから注釈を描画します。これにより、画像自体の荒れを防ぐとともに、文字や線のギザつきを抑えて美しく仕上げます。
* **アンチエイリアス線**: 斜めの注釈線を Pillow の標準描画からアンチエイリアス処理（内部4倍で描画してから縮小合成）へ置き換え、滑らかに描画します。
* **白黒モード**: 白黒出力オプションが有効な際、注釈線の色を白または黒から選べます（白に設定した場合、余白は視認性の高いグレーで表示されます）。
* **実験メモ**: 有効にすると、画像下部にテキストの実験メモを追加して出力します。

開始ライン、終了ライン、分子量マーカー、試料マーカー、ラベルは、レイヤーパネルのチェックボックス（📷マークの列）を外すことで出力画像から除外することができます。
なお、レイヤーパネルの目のマーク（👁マークの列）は操作画面上での表示・非表示の切り替えであり、出力画像には影響しません。

## 設定ファイル

ユーザー設定（言語など）は以下の場所に保存されます：

```
~/.EasyGelAlyzer/gel_config.json
```

このファイルは初回起動時に自動生成されます。
