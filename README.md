# EasyGelAlyzer

**EasyGelAlyzer** は、ゲル電気泳動画像から簡易的に定量解析を行う軽量なデスクトップアプリケーションです。タンパク質（SDS-PAGE）と DNA ゲルの両モードに対応しており、バンド位置の測定・Rf 値の算出・検量線フィッティング・分子量（または断片サイズ）の推定をひとつの GUI で完結できます。

\---

## 主な機能

* **デュアル解析モード** — タンパク質（SDS-PAGE、kDa）と DNA（アガロースゲル、bp）を選択可能
* **インタラクティブ画像ビューア** — ゲル画像のズーム・パン・回転、輝度・コントラスト調整
* **開始/終了ライン設定** — クリックまたはドラッグで泳動距離の基準ラインを配置
* **分子量マーカー登録** — マーカーバンドをクリックして既知サイズを入力すると検量線が自動更新
* **試料バンド測定** — 試料バンドをクリックすると検量線からサイズを自動推定
* **検量線フィッティング** — log₁₀(サイズ) vs. Rf の線形回帰と R² 表示；係数の手動上書きも可能
* **泳動ラインラベル** — ゲル画像上部にドラッグ可能なレーンラベル（MW マーカー／試料）を追加
* **レイヤーパネル** — アノテーション項目ごとに表示／出力可否をトグル；名前変更・並び替え・削除
* **Excel エクスポート** — マーカーデータ・試料結果・検量線グラフを多シート `.xlsx` ファイルに出力
* **アノテーション付き画像エクスポート** — 全アノテーションを描画した画像を保存；複数レイアウト対応
* **プロジェクト保存 / 読み込み** — 全計測データと画像を `.gelproj` JSON ファイルに保存して後日再開可能
* **アンドゥ / リドゥ** — 画像状態の 20 ステップ履歴
* **ドラッグ＆ドロップ** — 画像ファイルをキャンバスに直接ドロップ（`tkinterdnd2` 要）
* **日英バイリンガル UI** — 起動時に英語 / 日本語を切り替え可能

## インストール

### 方法 A — ビルド済み実行ファイルを使う（推奨）

[Releases](../../releases) ページから最新の `EasyGelAlyzer\_バージョン.zip` をダウンロードし、解凍後、`EasyGelAlyzer.exe` か `ショートカット` をダブルクリックで起動します。Python や追加ライブラリのインストールは不要です。

### 方法 B — ソースコードからビルドする

リポジトリをクローンして自分で実行ファイルを生成する場合：

1. **必要な環境を準備する**

   以下の環境が必要です：

   * **Python 3.9 以上**
   * **Windows（PyInstaller の仕様上）**

   必要な Python ライブラリをインストールします：

   ```bash
   pip install pillow numpy matplotlib openpyxl pyinstaller
   pip install tkinterdnd2   # ドラッグ＆ドロップ対応（任意）
   ```

2. **ビルドスクリプトを実行する**

   リポジトリのルートにある `exe\_build.bat` を実行します。

   * ダブルクリックで実行
   * またはターミナルから：

   ```bat
     exe\_build.bat
     ```

3. **ビルドスクリプトが行う処理**

   `exe\_build.bat` は以下の処理を自動で行います：

   * `src/version.py` からバージョン番号を読み取る
   * Git が利用可能な場合：

     * `git add .`
     * `git commit -m "Build vX.Y.Z"`
     * `git tag vX.Y.Z`
   * 日本語パスによる PyInstaller の不具合を避けるため、TEMP ディレクトリに一時ビルド環境を作成
   * PyInstaller を実行し、以下をバンドル：

     * `src/main.py`
     * `src/assets/`（アイコンなど）
   * 完成した `EasyGelAlyzer.exe` をローカルの `dist/` フォルダにコピー
   * `EasyGelAlyzer.lnk`（ショートカット）を自動生成
   * 一時ビルドフォルダを削除してクリーンアップ
4. **ビルド結果**

   ビルドが成功すると、以下が生成されます：

   * `dist/EasyGelAlyzer.exe`
   * `EasyGelAlyzer.lnk`（ショートカット）

   これらを使ってアプリを起動できます。

5. **注意事項**

   * `src/version.py` が存在しない場合、ビルドは中断されます
   * 日本語パス環境でも確実にビルドできるよう設計されています
   * `src/assets/` 内のアイコンは自動で exe に埋め込まれます
   * Git がインストールされていない場合はコミット・タグ付けをスキップします

## クイックスタート

1. **起動** — `EasyGelAlyzer.exe` をダブルクリックします（ソースからビルドする場合は先に `exe\_build.bat` を実行してください）。
2. **モード選択** — 起動ダイアログで「タンパク質（kDa）」または「DNA（bp）」を選びます。
3. **画像を開く** — *ファイル → 画像を開く* から、または JPEG / PNG / TIFF 形式のゲル画像をキャンバスにドラッグします。
4. **基準ラインの設定** — *開始ライン設定* をクリックしてゲルの上端をクリック、次に *終了ライン設定* で色素前線（または泳動の下端）をクリックします。
5. **マーカーを追加** — *マーカー追加* をクリックしてから各マーカーバンドをクリックし、既知サイズを入力します。
6. **検量線を確認** — 検量線パネルの R² 値を確認します（R² ≥ 0.95 が推奨）。
7. **試料を追加** — *試料追加* をクリックしてから各試料バンドをクリックすると、サイズが自動推定されます。
8. **エクスポート** — 表形式データは *エクスポート → Excel*、論文・報告書用図は *エクスポート → アノテーション画像* を使用します。
9. **保存** — *ファイル → プロジェクト保存* (`.gelproj`) で全データを保存し、後日再開できます。

## 検量線の計算方法

Rf 値は以下の式で算出されます：

```
Rf = (バンドの y 座標 − 開始ラインの y 座標) / (終了ラインの y 座標 − 開始ラインの y 座標)
```

次に線形回帰を行います：

```
log₁₀(サイズ) = a × Rf + b
```

未知サイズの推定は以下の式で行います：

```
サイズ = 10^(a × Rf + b)
```

マーカーがない場合は、検量線パネルで係数 `a` と `b` を手動入力することもできます。

## エクスポートの詳細

### Excel（`.xlsx`）

3 枚のシートが生成されます：

* **検量線シート** — マーカー名、Rf、サイズ、log₁₀(サイズ)
* **結果シート** — 試料番号、試料名、Rf、推定サイズ
* **グラフシート** — 検量線グラフ画像を埋め込み

### アノテーション付き画像

4 種のレイアウトから選択できます：

1. アノテーション付きゲル画像のみ
2. ゲル画像 + 結果テーブル
3. ゲル画像 + 検量線グラフ
4. ゲル画像 + テーブル + グラフ（一体型）

白黒モードおよび余白トリミングオプションも利用可能です。

## 設定ファイル

ユーザー設定（言語など）は以下の場所に保存されます：

```
\~/.EasyGelAlyzer/gel\_config.json
```

このファイルは初回起動時に自動生成されます。
